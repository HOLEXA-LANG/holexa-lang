# Changelog

## [1.0.0] - 2025-03-14

### Added
- Core language: variables, functions, structs, enums
- Type system: int, float, String, bool, Array
- Control flow: if/else if/else, while, for, loop, match
- String operations: concatenation, len, to_upper, to_lower, trim
- Built-in functions: println, print, input, len, exit, assert
- For-range loops: `for i in 0..10 { }`
- Inclusive ranges: `for i in 0..=10 { }`
- Pattern matching with `match`
- Recursive functions
- Compiler: hlxc (HOLEXA → C → GCC)
- Error system with HLX error codes
- Full test suite (30 tests, 0 failures)
